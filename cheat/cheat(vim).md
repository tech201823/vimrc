# vimチートシート
-[vimチートサイト](https://vim.rtorr.com/lang/ja/)<br>
上記を参考にした<br>
-編集中のコードを即時実行space+q<br>
-編集中のmdを即時実行space+o<br>
-nerdtree実行space+n<br>
-選択指定範囲vでヴィジュアルモード+カーソル移動<br> 
-vmodeで選択Space+rで置換<br>
## nerdtree
-nerdtree移動はvimと同じ<br>
-mでメニューが開く<br>
-ウィンドウ移動はctrl+w<br>

## snippet
-スニペット展開はctrl+e<br>
-スニペットはtabを使うと次の入力に移れる<br>

## neocmplchace
-展開はctrl+k<br>

## markdown
